# ASUTP Mixing Module: User Guide (EN)

This guide is the English version of the main user manual.

- Full content: `docs/user-guide.md`
- Recommended for operators and process engineers:
  - installation and first start,
  - daily workflow,
  - recipe revisions and changelog,
  - integration/export,
  - troubleshooting and update process.

For quick start:

1. Install with `asutp-mixing-module-setup.exe` (recommended), or use portable EXE.
2. Launch app and choose language/layout in sidebar.
3. Configure recipe + MaterialConfig.
4. Run simulation and monitor `HomogenizationViz`.
5. Validate/export payloads in `Integration`.
